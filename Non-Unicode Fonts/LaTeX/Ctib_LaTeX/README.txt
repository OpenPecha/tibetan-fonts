Package name:		cTibTeX (Tibetan for TeX and LaTeX2e)
Package authors:	Oliver Corff, based on work done by
			R. Schwartz, Jeff Sparkes, Sam Sirlin,
			Beat Steiner and Norbert Preining e.a.

Version/Release:	0.6

Features:		cTibTeX provides an environment for writing
			Tibetan on TeX and LaTeX2e systems without
			requiring any external preprocessor.


Installation:

emtex users: Read and follow instructions in EMTEX

MikTeX users: Read and follow instructions in MIKTEX

tetex users: Installation notes are contained in INSTALL
